Різновид використання селекторів

Селектори тегів(або селектори елементів) - це селектори з іменами тегів, які надають стилю відповідним HTML-елементам.

Приклад: блок з селектором span

            span {
                color: red;
            }

Якщо в селекторі вказати span, та задати значення наприклад колір color: red тоді всі елементи на сторінці з даним тегом <span> будуть червоного кольору, так як цей текст</span>.

Селектор класів - це можливість звертатися до групи різнорідних HTML-елементів які будуть належать до одного класу.

Приклад: блок з селектором span.color_blue

            span.color_blue {
                color: blue;
            }

Та вказати <span class="color_blue">текст буде синього кольору </span>.

Приклад: блок з селектором .color_green

            .color_green {
                color: green;
            }

Також без використання імені тегу <span class="color_green"> текст буде зеленого кольору</span>. Цей тип запису класу class="color_green" можливо застосовувати до різних тегів HTML сторінки.

Селектор id-імен за своїм використанням аналогічний до селектору класів, різниця лише в тім що селектор id-імен визначае HTML - тегі за id-іменами а не за класом.

Приклад: блок з селектором a#color_blueviolet

            a#color_blueviolet {
                color: blueviolet;
            }

Буде застосовано до HTML - тегів <a> у яких в атрибутах id вказано ім'я color_blueviolet, візьмем посилання: <a id=color_blueviolet> колір буде фіолетово-синій</a>.

Щоб використати селектор id-імен до інших HTML - тегів потрібно зробити наступний запис.

Приклад: блок з селектором без прив'язки до тегів #color_firebrick

            #color_firebrick {
                color: firebrick;
            }

Тепер цей id застосуємо до цього правила: <span id=color_firebrick>"Імена класів вразливі до регістру"</span>.

Груповий селектор - це коли один селектор відокремлений від другого комами та дозволяє застосувати стиль одночасно до всіх елементів які зазначені в груповому селекторі.

Приклад: блок з групових селекторів з тегів h2, h3, h4, h5

            h2, h3, h4, h5 {
                color: brown;
            }

<h2>Груповий селектор перший, коричневий</h2>

<h3>Груповий селектор другий, коричневий</h3>

<h4>Груповий селектор третій, коричневий</h4>

<h5>Груповий селектор четвертий, коричневий</h5>.

Універсальний селектор * застосовується для вибірки всіх елементів веб-сторінки. На цій сторінки застосовано універсальний селектор та застосовано стиль font-family: 'Times New Roman', Times, sans-serif;

Приклад: блок з селектором *

            * {
                font-family: 'Times New Roman', Times, sans-serif;
            }

Універсальний селектор часто використовують для обнуління властивостей marging і padding.

Вкладені селектори(селектор нащадків)складаються з декількох частин розділених пробілом, їх використовують щоб застосувати стилі до елементу якщо він вкладений в інший елемент.

Приклад: блок з селектором p span

            p span {
                color: coral;
            }

В даному випадку текст який оточений тегами елемента <span>знаходиться всередині <p>, <p><span>буде виділений кораловим кольором</span></p>,

на другі елементи <span> на веб-сторінки не вплине якщо не був вказаний другий стиль,<span>цей стиль буде такий як за значено вище червоним кольором</span>.

Псевдокласи визначають динамічний стан елементів, що змінюється за допомоги дій користувача.
Умовно всі псевдокласи діляться на три групи:

  • визначає стан елементів;
  • має відношення до дерева елементів(клас first-child);
  • вказує мову тексту(клас lang).

Псевдоклас визначаючи стан елементів належить одному з п'яти псевдокласів:

  • visited - відвідані
  • link - не відвідані
  • active - активні
  • focus - при отриманні фокуса
  • hover - на видимість який впливає курсор миші.

Приклад: блок з псевдокласом

            a:hover {
                color: red;
            }

Нижче застосовано пвевдоклас hover .

Застосовано - hover.
